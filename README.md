# Pixiepop

Una cabina de fotos virtual. Elige tu marco, toma 3 fotos y descarga tu tira.

## Demo

## Características

- Selector de marcos
- Cámara en tiempo real con cuenta regresiva
- Generación automática de tira de fotos
- Descarga PNG
- Soporte multiidioma (ES / EN)
- Diseño responsive

## Tecnologías
- Vue 3 + Composition API
- Pinia
- Vue Router
- Vue i18n
- PrimeVue
- Vite

## Instalación

```bash
npm install
npm run dev
```
